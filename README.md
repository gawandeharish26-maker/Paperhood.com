
# 📚 Paperhood – Print On Demand Landing Page

A modern, responsive landing page built  
The project showcases HTML, CSS, JavaScript, and UI/UX skills by creating a clean and user-friendly website for a print-on-demand book service.

----

## 🌐 Project Objective

Design and develop a **single-page responsive landing website** for a fictional print-on-demand platform called **Paperhood**.

The goal of this project was to demonstrate:
- Semantic HTML structure
- Responsive layout using CSS Flexbox/Grid
- Interactive functionality using Vanilla JavaScript
- Clean UI/UX design

----

## 🧠 Approach

The project was built from scratch using **pure HTML, CSS, and JavaScript** without any frameworks.


### Development Process

1. Planned page structure based on given task requirements
2. Created semantic HTML layout for accessibility and SEO
3. Built reusable CSS variables for theme and consistency
4. Designed responsive layout using Flexbox & Grid
5. Added JavaScript for interactivity and animations
6. Optimized for mobile, tablet, and desktop screens

The focus was on **clean code, reusable styles, and smooth user experience**.
